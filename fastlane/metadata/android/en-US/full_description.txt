Notifies you of the nearest station to your current location in real time.
It is useful for preventing missed stops on the train, checking your approach to your destination, and ensuring you don't miss scenic views from the train window.
Also, you can check information on closed stations (abandoned stations) that are not shown on standard maps. It is ideal for visiting abandoned stations and ruins.

<b>Key Features:</b>

- Uses device location information to notify you with vibration whenever the nearest station from your current location changes.
- Covers almost all existing stations in Japan, and data for some abandoned stations.

<b>Permissions:</b>

- Location (Always allow): Used for background movement detection and identifying the nearest station.
- Notifications: Used to inform the user of station information.
- Usage Access (Optional): Used to determine if a specific app (Station Memories!) is running in the foreground, and to suppress notification vibration while it is running.

<b>Background</b>

Released as an alternative following the suspension of updates for the "Nearest Station Search" (最寄り駅サーチ) app. This app uses the latest station information.

<b>About Station Data</b>

This app covers railway stations in Japan. It includes all existing stations and some closed stations.
This app utilizes the following data:

Seo-4d696b75/station_database : https://github.com/Seo-4d696b75/station_database

The user interface is available only in Japanese.
